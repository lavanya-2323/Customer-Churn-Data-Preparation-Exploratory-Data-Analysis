# Customer Churn Data Analysis

## 📌 Objective
To analyze telecom customer churn using data cleaning, feature engineering, 
and exploratory data analysis techniques.

## 📂 Dataset
Telco Customer Churn Dataset from Kaggle.

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## 🔄 Project Workflow
1. Data Understanding
2. Data Cleaning
3. Feature Engineering
4. Exploratory Data Analysis (EDA)
5. Insight Generation

## 📊 Key Insights
1. Month-to-month customers churn more.
2. High monthly charges increase churn.
3. Long tenure reduces churn.
4. Fiber optic users churn more.
5. Electronic payment users churn more.
